# YrkesCO Database Lab

## Project Description

This project contains the database design for YrkesCO.

The work includes:

Conceptual Data Model (CDM)
Logical Data Model (LDM)
3NF normalization
Physical Data Model (PDM)
SQL implementation
Sample queries

## Files Included

conceptual/ – Conceptual diagram and description
logical/ – Logical diagram and description
3NF/ – Normalization explanation
physical/ – Physical model diagram and description
YrkesCO.sql – Database table creation
queries.sql – SQL queries for testing

## Tools Used

dbdiagram.io
PostgreSQL
